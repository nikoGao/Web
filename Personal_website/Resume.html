Website for Log Searching and Data Analysis

Tsinghua University

The purpose of this project is to build up a website for users who want to analyze logs and collect useful information from each kind of log files. The basic function of this website is using logs generated by computer to analyze the system of each computer. My job in this team is using Java Maven to manage and share codes with other team member, using Bootstrap, HTML to build up front side of the website, using MySQL to manage all clients' information. To improve the management of logs uploaded by clients, we use cloud storage and basing on ElasticSearch, we could find the exact information hidden in logs more quickly.

“I CARE” Specialized System for Communication between Generations

Beijing University of Posts and Telecommunications

This is an innovation program proposed by our team. According to our investigations, we found out that in this modern and fast world, it's hard for offsprings to take care of their parents, so we want to design an APP for children and parents to directly contacting with each other. From the point of easy using, we designed two different interfaces for parents and children, simplified login methods for parents by utilizing a facial recognition system, and provided speech control between the various functions. Then we added functions like alerting parents' birthday, gifts recommendation and audio chat to promote the relationship between two generations.

Used techeniques: Java, Android Studio, Xunfei Audio Recognization API, Xunfei Face Recognization API
